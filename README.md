# Recipe Finder Website

## Overview

This Recipe Finder website is a college project developed using Vanilla HTML, CSS, and JS. The project aims to provide users with a simple and intuitive platform to discover and explore various recipes.

## Features

- **Search**: Users can search for recipes based on keywords, ingredients, or categories.
- **Filter**: Refine search results with filters such as cuisine, difficulty level, and cooking time.
- **Detailed Recipes**: Each recipe page includes detailed information, ingredients, and step-by-step instructions.
- **Responsive Design**: The website is designed to be responsive, ensuring a seamless experience across various devices.

## Technologies Used

- HTML
- CSS
- JavaScript
